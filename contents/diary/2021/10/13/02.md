### .DS_Storeとdot_clean

.DS_Storeを作らない設定

```
% defaults write com.apple.desktopservices DSDontWriteNetworkStores True
% killall Finder
```

.DS_Store と._xxx を一括削除する方法 1 コマンドを使って削除する

```
find . \( -name '.DS_Store' -or -name '._*' \) -delete -print
```
